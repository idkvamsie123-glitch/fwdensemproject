HTML & CSS Fundamentals

Build semantic HTML pages and style them with CSS including Flexbox, Grid, animations and responsive design.

HTML5CSS3FlexboxGridResponsive

📄 HTML Document Structure

Every HTML page follows a standard structure with a DOCTYPE declaration, html, head, and body elements. Semantic HTML5 elements like header, nav, main, section, article, and footer improve accessibility and SEO.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <meta name="viewport" content="width=device-width,initial-scale=1"/>
    <title>My Page</title>
  </head>
  <body>
    <header><nav>...</nav></header>
    <main><section>...</section></main>
    <footer>...</footer>
  </body>
</html>

🎨 CSS Selectors & Specificity

CSS selectors target HTML elements. Specificity determines which rule wins when multiple rules apply. Inline styles beat IDs beat classes beat elements.

/* Element selector — low specificity */
p { color: gray; }

/* Class selector */
.highlight { color: blue; }

/* ID selector — high specificity */
#hero { font-size: 2rem; }

/* Specificity: inline > ID > class > element */
/* 1000    >  100 >  10   >    1   */

📦 Box Model

Every HTML element is a box composed of: content, padding, border, and margin. Use box-sizing: border-box to include padding and border in the element's total width/height.

/* Box Model */
.box {
  width: 200px;         /* content width */
  padding: 20px;        /* inner space */
  border: 2px solid;    /* border thickness */
  margin: 16px;         /* outer space */
  box-sizing: border-box; /* width includes padding+border */
}

↔️ Flexbox Layout

Flexbox is a one-dimensional layout system for arranging items in rows or columns. It's perfect for navigation bars, cards, and centering elements.

.container {
  display: flex;
  justify-content: space-between;  /* horizontal alignment */
  align-items: center;             /* vertical alignment */
  gap: 16px;                       /* space between items */
  flex-wrap: wrap;                 /* wrap to next line */
}

.item {
  flex: 1;        /* grow equally */
  flex: 0 0 30%;  /* fixed 30% width */
}

⊞ CSS Grid Layout

CSS Grid is a two-dimensional layout system for complex page layouts. Define rows and columns explicitly and place items anywhere in the grid.

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 equal cols */
  grid-template-rows: auto;
  gap: 24px;
}

/* Responsive grid */
.grid {
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}

📱 Responsive Design

Responsive design adapts layouts to different screen sizes using media queries, relative units (%, em, rem, vw), and mobile-first approach.

/* Mobile first */
.hero { font-size: 1.5rem; }

/* Tablet */
@media (min-width: 768px) {
  .hero { font-size: 2.5rem; }
}

/* Desktop */
@media (min-width: 1024px) {
  .hero { font-size: 4rem; }
}
