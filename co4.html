DOM & Events

Manipulate the Document Object Model, handle user events, work with forms, and build dynamic interactive web pages.

DOMEventsFormsManipulation

🌳 What is the DOM?

The Document Object Model (DOM) is a tree-structured representation of the HTML document. JavaScript can read and modify the DOM to dynamically change content, structure, and styles without reloading the page.

document          ← root
 └─ html
     ├─ head
     │   └─ title → "My Page"
     └─ body
         ├─ h1 → "Hello"
         └─ p  → "World"

🔍 Selecting Elements

Use modern querySelector methods to select DOM elements. getElementById is fast for single elements by ID. querySelectorAll returns a NodeList of all matching elements.

// Single element
document.getElementById('myId');
document.querySelector('.class');
document.querySelector('#id');

// Multiple elements
document.querySelectorAll('.cards');
document.querySelectorAll('li');

// Navigate from element
el.parentElement;
el.children;
el.nextElementSibling;

✏️ Modifying the DOM

Once you select an element, you can change its text, HTML, attributes, styles, and CSS classes. Creating and appending new elements allows fully dynamic content generation.

const el = document.querySelector('#box');

// Change content
el.textContent = "New text";
el.innerHTML = "<strong>Bold</strong>";

// Modify styles & classes
el.style.color = "red";
el.classList.add('active');
el.classList.toggle('hidden');

// Create & append new element
const p = document.createElement('p');
p.textContent = "New paragraph";
document.body.appendChild(p);

🖱️ Event Listeners

Events allow JavaScript to respond to user actions like clicks, key presses, mouse movements, and form submissions. addEventListener is the modern way to attach events.

const btn = document.querySelector('#myBtn');

// Click event
btn.addEventListener('click', function(e) {
  console.log('Clicked!', e.target);
});

// Arrow function
btn.addEventListener('click', (e) => {
  e.preventDefault();     // stop default behavior
  e.stopPropagation();    // stop bubbling
});

// Keyboard event
document.addEventListener('keydown', (e) => {
  console.log(e.key, e.code);
});

📝 Form Handling

Access form values, validate inputs, and handle submission with JavaScript. Prevent default form submission to handle data with JavaScript instead of reloading the page.

const form = document.querySelector('#myForm');

form.addEventListener('submit', (e) => {
  e.preventDefault();  // stop page reload

  const name  = document.querySelector('#name').value.trim();
  const email = document.querySelector('#email').value.trim();

  if (!name) {
    showError('#name', 'Name is required');
    return;
  }

  console.log({ name, email });  // process data
});

💫 Event Delegation

Instead of adding event listeners to many elements, use event delegation: listen on a parent element and check which child was clicked. This is more efficient and works with dynamically added elements.

// ❌ Adding to each item (inefficient)
items.forEach(item => item.addEventListener('click', handler));

// ✅ Event Delegation (efficient)
document.querySelector('#list').addEventListener('click', (e) => {
  if (e.target.matches('.list-item')) {
    console.log('Item clicked:', e.target.textContent);
  }
  if (e.target.matches('.delete-btn')) {
    e.target.closest('.list-item').remove();
  }
});
