Master variables, functions, arrays, objects, ES6+ features, and asynchronous programming with Promises and async/await.

📦 Variables & Data Types

JavaScript has three ways to declare variables: var (function-scoped, avoid), let (block-scoped, reassignable), and const (block-scoped, constant reference). Data types include string, number, boolean, null, undefined, object, and symbol.

const name = "Alice";         // string
let   age  = 25;              // number
const isActive = true;        // boolean
let   data = null;            // null
let   x;                      // undefined

// Template literals (ES6)
const msg = `Hello, ${name}! You are ${age}.`;

⚡ Functions & Arrow Functions

Functions are reusable blocks of code. Arrow functions (ES6) provide concise syntax and lexically bind 'this'. Use default parameters and rest/spread operators for flexible functions.

// Traditional function
function greet(name) {
  return `Hello, ${name}!`;
}

// Arrow function
const greet = (name) => `Hello, ${name}!`;

// Default params & rest operator
const sum = (a, b = 0, ...rest) => {
  return a + b + rest.reduce((acc, n) => acc + n, 0);
};

📋 Arrays & Array Methods

Arrays store ordered lists. ES6+ array methods like map, filter, reduce, find, and forEach are essential for data transformation without mutating the original array.

const nums = [1, 2, 3, 4, 5];

nums.map(n => n * 2);          // [2, 4, 6, 8, 10]
nums.filter(n => n > 2);       // [3, 4, 5]
nums.reduce((a, n) => a + n);  // 15
nums.find(n => n === 3);       // 3

// Destructuring
const [first, second, ...rest] = nums;
// first=1, second=2, rest=[3,4,5]

🗂️ Objects & Destructuring

Objects store key-value pairs. Use dot or bracket notation to access properties. Destructuring, spread operator, and computed property names make objects easier to work with.

const user = { name: "Bob", age: 30, role: "admin" };

// Destructuring
const { name, age, role = "user" } = user;

// Spread to copy/merge
const updated = { ...user, age: 31 };

// Shorthand property names
const x = 10, y = 20;
const point = { x, y };  // { x: 10, y: 20 }

⏳ Promises & Async/Await

Asynchronous code handles operations like API calls without blocking. Promises represent eventual completion. Async/await is syntactic sugar over Promises for cleaner asynchronous code.

// Promise
fetch('/api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// Async / Await (cleaner)
async function getData() {
  try {
    const res  = await fetch('/api/data');
    const data = await res.json();
    return data;
  } catch (err) {
    console.error(err);
  }
}

🧹 ES6+ Features Summary

Modern JavaScript includes many powerful features: modules (import/export), classes, optional chaining, nullish coalescing, and more that make code cleaner and more robust.

// Optional chaining
const city = user?.address?.city;  // no error if undefined

// Nullish coalescing
const name = user.name ?? "Guest"; // fallback if null/undefined

// Modules
export const add = (a, b) => a + b;
import { add } from './math.js';

// Class syntax
class Animal {
  constructor(name) { this.name = name; }
  speak() { return `${this.name} makes a sound.`; }
}
